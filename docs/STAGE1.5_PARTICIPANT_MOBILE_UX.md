## 🎯 비인증 사용자 여정 검증

### 여정 단계

```
비인증 사용자
    ↓
초대 링크 클릭 (/join/[inviteCode])
    ↓ (JoinPage 로드 - 스크린샷 1 확인)
모임 정보 미리보기
    ↓
"로그인하여 참여 신청" 클릭
    ↓
로그인 페이지 (/auth/login)
    ↓ (LoginPage 로드 - 스크린샷 2 확인)
이메일 + 비밀번호 입력
    ↓
로그인 성공
    ↓
자동으로 /join/[inviteCode]로 복귀
    ↓
참여 신청 폼 표시
    ↓
참여 신청 제출
```

✅ **검증 완료**: JoinPage (초대 링크) → LoginPage (로그인) 여정 모바일 최적화 확인
